# hatci-audio-mixer
 
Synthetic audio generator for stress-testing automotive voice systems.
 
## Folder structure
- `docs/`: how-to guides
- `scripts/` installation scripts
- `demos/`: demo input files for testing
- `transform/`: transformation component for moving sound sources
 
## Quick start
See `docs/setup.md` for full setup instructions.

### Docker Compose

```
docker compose build
docker compose run --rm hatci-audio-mixer
```
